Chronic Kidney Disease (CKD) Classification:
Project Overview:
This project focuses on building a machine learning model to classify whether a patient has Chronic Kidney Disease (CKD) based on clinical features. The model was developed as part of the "Introduction to AI" course at Afeka College.
Key Features
The dataset includes 24 clinical features used for classification. Key steps in the development process included:
Data Preprocessing: Handled missing values using median and mode imputation to ensure data integrity. 
Feature Encoding: Prepared categorical and numerical data for model processing.
Algorithm Implementation: Utilized the Naive Bayes algorithm for efficient and probabilistic classification.
Results:
Accuracy: The model achieved an impressive accuracy rate of 98.75%.
Project Grade: Received a perfect score of 100.
Technologies Used:
Language: Python.
Libraries: pandas, matplotlib.pyplot, seaborn, sklearn.
